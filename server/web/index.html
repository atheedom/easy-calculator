To clear the calculator's results panel touch the C button.

To delete the last number entered touch the D button.

To calculate a percentage enter the fraction, like so: 10/100, and touch the % button.

The maximum number of numbers that can be displayed is 9.
This is a very simple and easy to use calculator designed for the Firefox OS running on mobile phones.
